

- Close all open files on disconnect

== ERROR ==
Result:
<errcode>    - 4 bytes
<command>    - 2 bytes
<message>    - 2 byte length + data + 1 byte '\0' terminator

== MKDIR ==
Request:

  <filename>  - 2 byte length + data + 1 byte '\0' terminator
Result:
  Error::OK  - 4 bytes
  <command>  - 2 bytes
